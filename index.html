Philosophie

Principes fondateurs

Zéro confiance

Nous ne demandons pas votre confiance, nous la rendons inutile. Le chiffrement s'effectue entièrement sur votre appareil. Même compromis, nos serveurs ne révéleraient rien.

Zéro connaissance

Nous ne savons rien de vous. Pas de compte, pas d'email, pas de numéro. Nous ne pouvons pas vous identifier, donc personne ne le peut.

Zéro trace

Fermez l'onglet, tout disparaît. Aucun historique, aucun log, aucune métadonnée. Les clés cryptographiques sont détruites à chaque session.

Transparence totale

Tout le code est ouvert et vérifiable. Faites un clic droit, affichez le code source. Pas de serveur caché, pas de tracking, pas de magie noire. Ce que vous voyez est ce qui s'exécute.
